# Dog Shelter Management System 🐾

A C++ application designed with a **layered architecture** to manage shelter operations. This project focuses on **Object-Oriented Programming (OOP)**.

## Key Features
- **Layered Architecture:** Clear separation between Domain, Repository, Service and UI layers.
- **Polymorphism:** Supports both In-Memory and File-Based storage of the repository.
- **Admin/User Modes:** Different permissions for managing dog profiles vs. viewing/adopting.

## Tech Stack
- **Language:** C++17
- **Build System:** CMake
- **Storage:** Text files 

## Future Enhancements
- Implement a Graphical User Interface (GUI) using **Qt**.
